How to use the Box Editor

To begin, upload an image or use the provided sample. Select the buttons in the toolbar to perform certain actions such as identifying lines of text and generating suggestions.

Once text boxes are generated for all lines of text, you can start editing as needed. Either manually type in the box dimensions in the coordinate fields or drag the handles on the image to change the size or position of a text box. Use the tools on the right to split boxes into columns, draw new ones, or delete existing boxes.

When the document is complete, download the .box file and .gt.txt files and use them to train your OCR model.

The progress bar shows the progress of the tagging. When done, download the .box file and the .gt.txt file to use in Tesseract.

About

This is a powerful Box Editor for Tesseract allowing for efficient generation, editing, and managing text boxes in WordStr format. The tool uses a custom trained model for authoring the Romanian Transitional Script.
